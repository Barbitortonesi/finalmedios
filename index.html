¿Que es y para que sirve?

Filezilla es una aplicación que se conecta a un servidor FTP para gestionar y acceder a los archivos. Y un servidor FTP es otra aplicación -o servicio- que usa el protocolo FTP (File Transfer Protocol) para compartir archivos con otros usuarios. Por tanto, FileZilla sirve para subir y descargar archivos de un FTP, así como para realizar tareas comunes con archivos igual que lo harías en el explorador de archivos de tu PC.
